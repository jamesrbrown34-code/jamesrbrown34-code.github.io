# TubeRefund Ltd

## About

TubeRefund is a subscription-based browser extension that helps users identify and manage potential public transport delay refund claims.

The software analyses a user's journey history and highlights potential eligible delay refund claims. Paid users receive additional automation features to assist with completing refund forms. All claims are reviewed and submitted directly by the user.

TubeRefund Ltd is not affiliated with Transport for London (TfL) or any other transport authority.
